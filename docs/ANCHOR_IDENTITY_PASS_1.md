# Anchor Identity Pass 1

This pass sharpens the named core faces of the current pool so they feel as intentional as the newer support shell from Bridge Set 0.

## Summary

- Revised: `Hamlet`, `Alice`, `Lady Macbeth`, `Queen of Hearts`
- Added: `The Annotator` as a durable knowledge payoff
- Did not add a separate extra sustain/protection card in this pass

Note:
`The Annotator` was later removed from the live pool and replaced by `Close Reading`, a Neutral payoff plot that serves the same knowledge-support slot with a better flavor fit and lower trigger complexity.

The reasoning for skipping an extra sustain card was simple: `Stay of Execution` already gave the pool a readable post-damage support tool, and this pass needed to spend its complexity budget on making the main named cards actually feel like anchors first.

## Card Changes

- `Hamlet`
  Old role: efficient pressure filler
  New role: reveal/knowledge engine
  Archetypes supported: `Shakespeare Pressure Defeat`, `Knowledge Control Midrange`
  Function: `engine`
  Why: he now reads the enemy's hand and turns reflection into knowledge, which better matches the character and gives Shakespeare a more deliberate line

- `Alice`
  Old role: resilient Carroll body
  New role: reveal-to-tempo payoff
  Archetypes supported: `Carroll Tempo Reveal`
  Function: `payoff`
  Why: she now rewards the player for entering the Wonderland information game instead of just being a sturdy body

- `Lady Macbeth`
  Old role: generic midcurve stat stick
  New role: pressure/debuff bridge
  Archetypes supported: `Shakespeare Pressure Defeat`, `Debuff Control`
  Function: `bridge`
  Why: she now manipulates the enemy's best unit immediately, which gives Shakespeare pressure decks a clearer short-window play pattern

- `Queen of Hearts`
  Old role: plain aggressive body
  New role: volatile reveal-to-pressure payoff
  Archetypes supported: `Carroll Tempo Reveal`
  Function: `pressure piece`
  Why: she now converts exposed information into direct writer damage, which feels much more like Wonderland authority turned into sudden punishment

## Historical Durable Knowledge Payoff

- `The Annotator`
  Author: `Neutral`
  Type: `character`
  Tags: `knowledge`, `tempo`
  Function: `engine/payoff`
  Rules role: if you reach 3+ Knowledge, the card starts paying you back every turn with extra draw

This was designed as a character rather than another plot because the knowledge lane already had setup and one-shot payoff. What it lacked was a visible board piece that tells the player, "if you keep the knowledge engine running, this will keep mattering."

## Why No Extra Sustain Card

- `Stay of Execution` already exists as a readable post-damage preservation tool.
- The bigger need in the live pool was card identity clarity on the named faces, not another generic defensive support effect.
- Pre-defeat protection is still a valid future target, but it should be added only if match testing still shows slower shells collapsing too fast.

## Expected Archetype Impact

- `Carroll Tempo Reveal` should now have both a clearer engine body (`Alice`) and a clearer payoff body (`Queen of Hearts`).
- `Shakespeare Pressure Defeat` should now have a stronger thought-to-action anchor (`Hamlet`) and a cleaner manipulation bridge (`Lady Macbeth`).
- `Knowledge Control Midrange` should now feel more like a real deck because it finally has a clear payoff slot instead of only temporary payoff moments.

## Balance Watchpoints

- `Hamlet` may become too efficient if a cost-1 Shakespeare play pattern plus free knowledge proves too easy to snowball.
- `Queen of Hearts` needs testing to make sure reveal-enabled writer damage feels rewarding rather than too bursty.
- The later `Close Reading` replacement should be tested to confirm whether its threshold draw payoff is clearer and less noisy than the old Annotator body.
